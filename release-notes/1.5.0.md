# UniGeek v1.5.0

## What's New

> **Achievement System**
- 60+ achievements across 10 domains: WiFi, BLE, Keyboard, NFC, IR, Sub-GHz, GPS, Utility, Games, Settings
- Earn EXP and level up your rank
- Unlock an **Agent Title** to display on your Character Screen

> **Character Screen**
- Full-screen profile showing rank, EXP, HP, BRAIN, domain progress bars, and your Agent Title

> **BLE Device Spam** (new submenu under BLE)
- **Android** — Google Fast Pair popup (31 model IDs, randomized MAC)
- **iOS** — Apple Continuity popups: SourApple action popups + AppleJuice AirPods/setup popups
- **Samsung** — Galaxy Watch pairing popup (26 watch models)

> **New Games**
- **Memory Sequence** — Simon Says-style memory game, 4 difficulty levels, with high score tracking
- **Number Guess** — Higher/lower number guessing game, 4 difficulty levels (1–99 to 1–9999)

> **High Scores**
- Top 5 scores per difficulty tracked for: Wordle, HEX Decoder, Number Guess, Memory Sequence, Flappy Bird

> **Karma Attack**
- Karma EAPOL and Karma Captive are now separate modes
- Karma Support: companion mode for a second device to host fake WPA2 APs via ESP-NOW

## Improvements
- Download menu now includes BadUSB scripts by OS and category
- Better EAPOL offline brute-force
- Simplified sprite rendering across screens

## Bug Fixes
- PSRAM heap allocation
- I2C detector and pin settings
- Font position and dialog bubble rendering
- WiFi connect stability
- IP ping fix
- IRAM usage reduction
