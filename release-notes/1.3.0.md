# UniGeek Firmware v1.3.0

## New Features

> **NFC (MFRC522)**
- MIFARE Classic card reader and key recovery tool
- Scan UID — detect card UID and type
- Authenticate — test all sectors with common default keys
- Discovered Keys — view recovered keys per sector
- Dump Memory — read all card data using discovered keys
- Dictionary Attack — try keys from custom dictionary files
- Static Nested Attack — recover keys on cards with static nonce
- Darkside Attack — recover the first key when no keys are known

> **GPS & Wardriving**
- Live GPS view with coordinates, speed, altitude, satellite count, and UTC time
- WiFi + BLE wardriving with GPS coordinates in Wigle-compatible CSV format
- NimBLE BLE scanning works on all boards (no PSRAM required)
- Promiscuous WiFi capture with channel hopping across all 13 channels
- Scrolling log display with WiFi count, BLE count, distance, and elapsed time
- GPS time sync to device clock (and RTC where available)

> **Wigle Integration**
- Upload wardrive logs to Wigle from GPS module or Network menu
- View Wigle profile stats: rank, WiFi/Cell/BT discovered, upload history
- Manage API token — shared between GPS and Network > Wigle
- Uploaded files marked with "Uploaded" status and renamed with _uploaded suffix

> **Barcode Generator**
- Code 128 barcode display from typed text or file
- Load barcode content from /unigeek/barcode/ on storage
- Inverted color toggle (UP/DOWN while viewing)

> **BLE Detector**
- BitChat app user detection added to passive BLE scanning

> **Pin Configuration**
- Configure external I2C SDA/SCL pins from Settings
- Configure GPS TX, RX, and baud rate from Settings
- T-Lora Pager GPS pins pre-configured

## Bug Fixes

- Power saving now respects blocking action overlays — screen won't turn off during popups or input dialogs
- ScrollListView flicker eliminated
- Partition size increased for more app storage
- Cardputer keyboard logic error fixed
- I2C detection fix for MFRC522 module
- T-Lora Pager hardware fixes
- Web file manager fix

## Supported Devices

- M5StickC Plus 1.1
- M5StickC Plus 2
- LilyGO T-Lora Pager
- LilyGO T-Display 16MB
- DIY Smoochie
- M5Stack Cardputer
- M5Stack Cardputer ADV
